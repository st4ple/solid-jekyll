More About Sebastian Zanlongo.

I am a Teaching Assistant for Java Programming II. Past classes include Intro to Microcomputers, Java Programming I and II, and Principles of Programming Languages.

Detail-oriented and able to clearly communicate highly technical issues orally and in writing. Designed material to address the needs of diverse adult learners.
