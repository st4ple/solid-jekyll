Class Tips

Doing well in the Course
  • Attend class - Material covered in class will be implemented in labs.
  • Attend labs - Work done in labs will provide a framework for your assignments.
  • Do assignments on your own - these will serve as inspiration for your exam questions.
Learning Java
  • Learn to "play computer" - be able to hand-trace program execution.

Java Tricks

Shortcuts
  • Tired of writing System.out.println()? Next time type sout and press Tab.
  • Don't format code by hand. Instead, right-click and select Format, or press Alt+Shift+f.
  • Avoid scrolling through lots of code by using the Outline pane to navigate.
  • Assigning a value based on an if-else? Try replacing: 
    if (a > b) {
  max = a;
}
else {
  max = b;
}
     with this: max = (a > b) ? a : b;
Debugging
  • // comments help document your program. This is great for you to review later, and to ensure proper grading. A quick way of commenting lines of code is to highlight them and then hit ctrl+/.
