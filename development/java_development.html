Setting Up IDE

There are many different Integrated Development Environments. Below are some of the ones used in previous classes:

Netbeans (Recommended for Class)

1. Download Java

2. Download JDK + Netbeans

Eclipse (I use this!)
Bundled installer for Java, JDK, and Eclipse: (x86) or (x64).
jGRASP

1. Download Java + JDK x86 or x64

2. Download jGrasp
