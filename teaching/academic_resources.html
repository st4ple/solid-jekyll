Academic Discounts

Revision Control

I find that a lot of students carry their code on a thumb drive. This is acceptable for Programming I and II, however you will find that this becomes unwieldy for more complicated projects. GitHub has a great list of reasons to use version control. Below are some recommendations:

Repositories

These are the places that will host your code:

  • GitHub is best for showing off public work. You can also grab an academic license which gives you 5 private repos.
  • Bitbucket freely provides unlimited private repos (great for classwork).
Clients

Getting started with git can be daunting, and many people prefer using a GUI. In these cases, my personal preferences are:

  • SourceTree for Windows and Mac
  • SmartGit for Linux

Free Software

Remember to use the resources your university provides you! In the case of FIU, you have:

  • ShopUTS, which has things like Microsoft Office
  • Microsoft DreamSpark, which has Microsoft Windows, Visual Studio, etc.

Question and Answer

The Stack Exchange network is a fantastic resource for questions in almost any area you can think of. Chances are your question has already been answered! Some of the more relevant communities are:

  • Computer Science - students, researchers and practitioners of computer science
  • Mathematics - people studying math at any level and professionals in related fields
  • Programmers - professional programmers interested in conceptual questions about software development
  • Robotics - professional robotic engineers, hobbyists, researchers and students
  • Super User - computer enthusiasts and power users
  • Theoretical Computer Science - theoretical computer scientists and researchers in related fields
  • Stack Overflow - professional and enthusiast programmers
